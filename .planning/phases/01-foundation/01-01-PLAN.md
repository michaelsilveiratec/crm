# Plan: 01-01-FOUNDATION

**Phase:** 1 (Foundation)
**Goal:** Estabelecer a infraestrutura básica com Go, Preact e SQLite.
**Status:** Ready to execute

## Context
Este plano foca na "Fase 1: Fundação" do ROADMAP.md. O objetivo é ter um servidor Go rodando, servindo uma interface Preact minimalista e conectando ao SQLite.

## Requirements Covered
- **INFRA-01**: Persistência de dados em SQLite.
- **INFRA-02**: Binário único em Go que serve o frontend Preact embutido.

## Verification Criteria
1. Ao rodar `go run ./cmd/server/main.go`, o servidor deve iniciar na porta 8080.
2. Abrir `http://localhost:8080` no navegador deve mostrar a interface básica do Preact.
3. Chamar `http://localhost:8080/api/health` deve retornar um JSON de sucesso.
4. O arquivo `crm.db` deve aparecer na raiz do projeto.

## Final Deliverable
Binário único (emulado via `go run`) rodando backend, frontend e banco de dados.
