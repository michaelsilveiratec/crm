# Sketch 001: Navigation & Layout

## Design Question
Como o usuário deve navegar entre as diferentes áreas do sistema?

## How to View
Abra o arquivo `.planning/sketches/001-navigation-layout/index.html` no seu navegador.

## Variants
- **A: Sidebar (Clássico)** — Uma barra lateral fixa à esquerda. Comum em sistemas desktop/gestão.
- **B: Top Nav (Moderno/Mobile)** — Navegação no topo, liberando mais espaço horizontal. Ideal para uso em celulares.

## What to Look For
Compare como o espaço é aproveitado e qual parece mais intuitivo para o uso diário na igreja.
