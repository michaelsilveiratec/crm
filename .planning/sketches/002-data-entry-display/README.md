# Sketch 002: Data Entry & Display

## Design Question
Como facilitar a entrada de dados para o obreiro e a visualização rápida para o pastor?

## How to View
Abra o arquivo `.planning/sketches/002-data-entry-display/index.html` no seu navegador.

## Variants
- **A: Cadastro (Obreiro)** — Focado em inputs claros, agrupados e fáceis de preencher no celular.
- **B: Gestão (Pastor)** — Lista limpa com ações rápidas (botão WhatsApp verde) e badges de status (aniversário, novos).

## What to Look For
Veja se os campos do formulário estão na ordem correta e se a lista de visitantes do pastor permite tomar ações rápidas (como enviar boas-vindas).
