# Sketch Manifest

## Design Direction
Um CRM eclesiástico extremamente leve, moderno e focado em dispositivos móveis. O visual deve ser limpo, transmitindo confiança e facilidade de uso tanto para o obreiro (cadastro rápido) quanto para o pastor (gestão clara).

## Reference Points
- Modern CRM (Clean dashboards)
- Google Forms (Simple inputs)
- Pico.css (Lightweight aesthetic)
